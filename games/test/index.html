Dice Transform Test

This test systematically checks each dice face transform to identify incorrect positioning.

Test Results

Click "Start Face Transform Test" to begin testing each face orientation.

How to Use This Test:

  1. Click "Start Face Transform Test"
  2. The test will cycle through 6 face orientations
  3. For each orientation, check:
    • Which face the algorithm detects (shown in results)
    • What dots you visually see on the dice
    • Whether they match
  4. Results will show any mismatches between expected and detected faces
